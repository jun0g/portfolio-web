# Personal Career Web Page

## Overview
이 프로젝트는 개인 커리어를 체계적으로 정리하고 공유하기 위한 웹 페이지이다.

기존 이력서 형식에서는 전달하기 어려운 프로젝트 경험, 문제 해결 과정, 기술 실험 기록 등을 보다 상세하게 정리하고 제공하는 것을 목표로 한다.

또한 구직 활동 시 이력서의 보조 자료로 활용하여, 프로젝트 경험과 기술적인 접근 방식, 문제 해결 과정을 보다 구체적으로 전달하기 위한 목적으로 제작되었다.

---

## Purpose

- 커리어 전반의 흐름을 한눈에 확인할 수 있도록 정리
- 주요 프로젝트 경험과 역할을 상세히 기록
- 실제 업무 중 발생한 문제 해결 과정을 Work Log 형태로 기록
- DevOps 관련 기술 실험 및 연구 내용을 Lab 형태로 정리

---

## Sections

### Profile
기본적인 이력서 정보를 제공하는 섹션

포함 내용
- 자기소개
- 경력 및 학력
- 기술 스택
- 자격증
- 어학
- 성장 과정 및 향후 포부
- 지원 동기

---

### Projects
주요 프로젝트 경험을 정리한 섹션

포함 내용

- 프로젝트 개요
- 프로젝트 기간
- 담당 역할
- 사용 기술
- 주요 수행 내용
- 성과 및 결과

---

### Work Log
업무 중 발생했던 이슈 해결 및 트러블슈팅 기록

포함 내용

- 문제 상황
- 발생 원인
- 해결 과정
- 적용 결과
- 회고

---

### Lab
DevOps 관련 기술 실험 및 개인 연구 내용을 정리하는 섹션

포함 내용

- 실험 목적
- 실험 환경
- 실험 과정
- 결과 및 검증
- 운영 적용 가능성
- 한계 및 개선 방향
